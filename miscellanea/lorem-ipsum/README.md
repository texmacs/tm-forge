Lorem Ipsum generator for TeXmacs
=================================

Wikipedia page on [Lorem ipsum](https://en.wikipedia.org/wiki/Lorem_ipsum).

[Example PDF output](./test.pdf)

Use
--------

You can just symbolically link this directory is in ~/.TeXmacs/plugins, e.g. with (assuming your current directory is the right one)
```shell
pushd ~/.TeXmacs/plugins
ln -s $PWD lorem-ipsum
popd
```
The package `lorem-ipsum` will then be available inside TeXmacs. Use [test.tm](./test.tm) as test document.
